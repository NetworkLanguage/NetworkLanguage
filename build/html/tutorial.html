3. 安装环境及代码使用说明¶

3.1. 安装环境¶

  • Python 3 +
  • Pytorch 1.1
  • Pandas
  • Numpy
  • jieba
  • tqdm
  • sklearn
  • gensim
  • fasttext
  • tensorboard

3.2. 使用说明¶

数据集选择¶

在每个方法对应的代码中取消注释以选择相应的数据集

# task = "weibo_gender"
# task = "weibo_area"
task = "douban"

BOW¶

  • code
  • 训练并测试
python bow.py

Word2vec¶

  • code
  • main函数中可调整参数有：
    • dim: 词向量维度
    • iter: word2vec训练迭代次数
  • 训练并测试
python w2v.py

fastText¶

  • code
  • main函数可调整参数有：
    • dim: 特征维度
    • epoch：模型训练迭代次数
  • 训练并测试
python fast_text.py
